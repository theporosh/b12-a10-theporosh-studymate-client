## Project name (client) : b12-a10-theporosh-studymate-client
## Project name (server) : b12-a10-theporosh-studymate-server

 ### Purpose : StudyMate is a MERN Stack web platform designed to help students connect and collaborate for better learning outcomes.It enables users to find study partners based on subjects, learning preferences, or nearby locations, making education more interactive, engaging, and goal-oriented.

 ### live URL Client: https://b12-a10-poroshstudymate-client.web.app
 ### live URL Server: https://b12-a10-theporosh-studymate-server.vercel.app/

 ### key features: In this skillswap app you can learn many skill 
 1. Students can find partners in find partners option
 2. Students can create Partner Profile 
 3. Students also can view details partner profile and also make a request after they register in application
 4. In My Connection section student can see their own connections and can update and delete if they want
 5. After he/she able to successfully login/register student can see their own profile in right side if they click on photo.
